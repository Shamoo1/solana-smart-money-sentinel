# Smart Money Entry Alert - Solana Monitor

Real-time monitoring dashboard for Pump.fun and Bags token launches on Solana with smart money filtering criteria.

## Framework

Vite 6.x + React 19.x

## Features

- Real-time token monitoring from Pump.fun and Bags
- Smart money entry criteria filtering:
  - Age: Within 20 minutes
  - Market Cap: $5K–$10K
  - Liquidity: ≥$10K
  - Bundles: ≤20%
  - Snipers: ≤20%
  - Insiders: ≤20%
  - Pro Traders: ≥15%
  - Dev Holding: ≤8%
- Visual and sound alerts for matching tokens
- Customizable filter parameters
- Clean, modern UI with real-time updates

## Preview

Configured to run on `0.0.0.0:8080` for k8s HTTPRoute access.

## Theme

Clean Light theme with professional blue accents, optimized for financial data display.
